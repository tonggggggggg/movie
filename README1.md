# 線上電影票購買系統的設計與實現
**11224137鄭宇彤 11224121黃靖雅**

## 一、執行目的
像是最近上映爆紅的電影，製作線上電影票購買系統，讓大家可以不用花時間去現場排隊，並以軟體工程方法進行系統分析與設計，模擬實際線上訂票平台之運作流程，提供電影瀏覽、場次選擇、座位選擇、訂單管理與付款等功能，然後重點放在系統架構與設計文件。

## 二、專案目標
設計並實作一套「線上電影票購買系統」，讓使用者可以：

* 瀏覽電影資訊
* 選擇場次與座位
* 線上訂票並完成付款
* 查詢與管理訂單

## 三、系統特色

 - 採用前後端分離概念（RESTful API）

 - 多層式架構（Controller / Service / Repository）

 - 支援電影、場次、座位、訂單與付款流程

 - 系統流程清楚，符合實際購票情境

 - 架構設計完整，適合中大型系統規劃與擴充

## 五、主要功能模組
1.電影管理

* 電影資料新增 / 編輯 / 下架

* 電影資訊查詢（片名、簡介、片長、級別）

 2. 場次管理

* 電影場次排程

* 影廳與播放時間設定

* 場次狀態管理

3.座位管理

* 座位配置顯示

* 即時座位狀態更新（可選 / 已售）

* 避免重複售出座位

4. 訂單與購票流程

* 選擇電影、場次與座位

* 建立訂單

* 訂單狀態管理（未付款 / 已付款 / 取消）

5.付款模組（模擬）

* 模擬付款成功或失敗

* 更新訂單與座位狀態

6. 訂單查詢

* 使用者查詢歷史訂單

* 顯示訂票明細與座位資訊

7.系統管理與安全

* 使用者登入與身分識別

* 管理員權限控管

* 系統操作日誌（Audit Log）

* 系統通知（購票成功、付款結果）
